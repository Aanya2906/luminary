# 🌌 Luminary
Luminary is a mood-based ambient focus environment that transforms your emotional state into a living, interactive universe.
It blends visual aesthetics, soundscapes, and productivity tools to help you focus, reflect, and create.

---

## ✨ Features
* 🎭 **8 Emotional Frequencies**
  Switch between moods like *Serene, Focus, Create, Calm, Energy, Reflect, Release, Wonder*

* 🌈 **Dynamic Color Resonance**
  UI adapts in real-time based on your selected mood

* 🌌 **Generative Particle System**
  A living visual field that reacts to your emotional state

* 🎧 **Ambient Soundscapes**
  Example: *Rain on glass* for deep focus

* ⏱️ **Pomodoro Timer**
  Stay productive with structured focus sessions

* 📝 **Thought Journal**
  Capture ideas and track your mental state
